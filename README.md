## 源阅 - 浏览器


### 介绍

源阅是一个浏览器工具，主要用于提升浏览内容的浏览体验。

#### 干净

* 无广告、无推送、无新闻。

#### 自定义视频播放器

* 支持滑动快进、快退、调整音量、明暗功能。
* 支持倍速播放功能。
* 支持锁屏状态横屏全屏播放。
* 支持投屏功能，远程遥控功能。

#### 过滤干扰阅读的元素

* 提供了基础的广告过滤开关。
* 提供了手动标记广告功能，能有效过滤干扰阅读的网页内容。

#### 智能阅读模式

* 能根据网页内容生成阅读模式，提供更优良的阅读体验。

#### 多格式阅读器

* 支持txt、mobi、epub、azw3等多种格式本地书籍。

#### 语音朗读

* 支持多种平台TTS语音朗读。
